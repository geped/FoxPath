L'assistente visivo per i tuoi percorsi Windows.

Recupera, formatta e gestisci file e cartelle con una simpatica mascotte interattiva. Progettato per la produttività, con stile.

Scarica FoxPath ahk_v2_ready

Stati della Mascotte

STARTUP

La Volpe ti accoglie istantaneamente ogni volta che la evochi con la scorciatoia globale.

WAITING

Sempre vigile, in attesa di rilevare una nuova selezione su Esplora Risorse o Desktop.

SUCCESS

Feedback visivo immediato non appena il percorso viene elaborato e copiato.

Funzionalità Principali

CORE

Rilevamento Intelligente

Legge in automatico il file o la cartella selezionata su Esplora File o Desktop per offrirti solo le opzioni adatte al contesto.

NEW

DevTools Integrati </>

Una suite nativa a portata di clic: codifica Base64, Hash MD5/SHA, Tester Regex, Convertitore Timestamp e formattazione JSON.

MEDIA

Immagini e Colori

Color Picker visivo per estrarre le 5 tinte predominanti dalle foto e un convertitore di formato ultra-rapido (PNG/JPG/BMP).

TEXT

Estrazione Testo & OCR

Estrai testo e tabelle da Word/Excel senza aprirli, o usa il motore per estrapolare stringhe da intere immagini.

POWER

Super Tools

Comprimi file in archivi multipli, camuffa data e ora, rinomina elementi in blocco e distruggi file in modo irrecuperabile (Shredder).

CODE

Codice e Snippet

Apri al volo in VS Code o Terminale, gestisci repo Git e genera boilerplate pronti all'uso (HTML, Markdown, gitignore).

Architettura & Sistema

foxpath_config.ini

; Impostazioni salvate automaticamente

GlobalTransparency = 200

LockAspectRatio = true

GlobalHotkey = ^!f

SoundEffects = enabled

FoxPath è scritto interamente in AutoHotkey v2. Utilizza GDI+ per il rendering delle immagini e un sistema di hooking per monitorare le finestre attive in tempo reale.

Come usarlo

  1. 01
    Evoca la Volpe: Premi Ctrl + Alt + F in qualsiasi momento.
  2. 02
    Seleziona un Elemento: Clicca su uno o più file sul Desktop o in Esplora Risorse.
  3. 03
    Copia & Formatta: Usa il tasto sinistro per la copia standard o il destro per le opzioni avanzate.
