Main Features

CORE

Smart Detection

Automatically reads the selected file or folder on File Explorer or Desktop to offer only context-appropriate options.

NEW

Integrated DevTools </>

A native suite at your fingertips: Base64 encoding, MD5/SHA Hash, Regex Tester, Timestamp Converter, and JSON formatting.

MEDIA

Images and Colors

Visual Color Picker to extract the 5 predominant colors from photos and an ultra-fast format converter (PNG/JPG/BMP).

TEXT

Text Extraction & OCR

Extract text and tables from Word/Excel without opening them, or use the engine to extract strings from entire images.

POWER

Super Tools

Compress files into multiple archives, disguise date and time, batch rename items, and irretrievably destroy files (Shredder).

CODE

Code and Snippets

Open on the fly in VS Code or Terminal, manage Git repos, and generate ready-to-use boilerplates (HTML, Markdown, gitignore).

Architecture & System

foxpath_config.ini

; Settings automatically saved

GlobalTransparency = 200

LockAspectRatio = true

GlobalHotkey = ^!f

SoundEffects = enabled

FoxPath is written entirely in AutoHotkey v2. It uses GDI+ for image rendering and a hooking system to monitor active windows in real time.

How to use it

  1. 01
    Summon the Fox: Press Ctrl + Alt + F at any time.
  2. 02
    Select an Item: Click on one or more files on the Desktop or in File Explorer.
  3. 03
    Copy & Format: Use the left click for standard copying or the right click for advanced options.
