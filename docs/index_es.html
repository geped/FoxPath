FOXPATH
v2.1 RELEASED

El asistente visual para tus rutas de Windows.

Recupera, formatea y gestiona archivos y carpetas con una simpática mascota interactiva. Diseñado para la productividad, con estilo.

Descargar FoxPath portable_exe_ready

Estados de la Mascota

STARTUP

El Zorro te saluda instantáneamente cada vez que lo invocas con el atajo global.

WAITING

Siempre vigilante, esperando detectar una nueva selección en el Explorador de Archivos o Escritorio.

SUCCESS

Retroalimentación visual inmediata tan pronto como la ruta es procesada y copiada.

Características Principales

CORE

Detección Inteligente

Lee automáticamente el archivo o carpeta seleccionada en el Explorador de Archivos o Escritorio para ofrecerte solo las opciones adecuadas al contexto.

NEW

DevTools Integrados </>

Una suite nativa a tu alcance: codificación Base64, Hash MD5/SHA, Probador Regex, Convertidor Timestamp y formato JSON.

MEDIA

Imágenes y Colores

Selector de Color Visual para extraer los 5 colores predominantes de fotos y un convertidor de formato ultrarrápido (PNG/JPG/BMP).

TEXT

Extracción de Texto y OCR

Extrae texto y tablas de Word/Excel sin abrirlos, o usa el motor para extraer cadenas de imágenes completas.

POWER

Super Herramientas

Comprime archivos en múltiples archivos, camufla fecha y hora, renombra elementos por lotes y destruye archivos de forma irrecuperable (Shredder).

CODE

Código y Snippets

Abre al instante en VS Code o Terminal, gestiona repositorios Git y genera plantillas listas para usar (HTML, Markdown, gitignore).

Arquitectura y Sistema

foxpath_config.ini

; Configuraciones guardadas automáticamente

GlobalTransparency = 200

LockAspectRatio = true

GlobalHotkey = ^!f

SoundEffects = enabled

FoxPath está escrito íntegramente en AutoHotkey v2. Utiliza GDI+ para el renderizado de imágenes y un sistema de enlace (hooking) para monitorizar las ventanas activas en tiempo real.

Cómo usarlo

  1. 01
    Invoca al Zorro: Presiona Ctrl + Alt + F en cualquier momento.
  2. 02
    Selecciona un Elemento: Haz clic en uno o más archivos en el Escritorio o en el Explorador de Archivos.
  3. 03
    Copiar y Formatear: Usa el clic izquierdo para la copia estándar o el derecho para opciones avanzadas.
